## Description

코드 디버깅을 통해 모든 JUnit Test를 통과해보는 프로젝트입니다.

해당 프로젝트는 다음의 목적을 갖습니다.
- Maven Project를 import, update 해보기
- 컴파일 에러 고치기
- JUnit Test 통과를 위한 요구사항 충족시키기
- Java언어에 대한 더욱 깊은 학습

## Project Setting

- Eclipse Project with Maven
- JUnit 5.7.2
- JDK 1.8

## Chapters

- Collection
- Object `equals`, `hashCode`
- Exception
- FileIO
